# 第二部分 学会外语的 5 项核心原则

要掌握任何领域的知识和技能，先认识核心原则是最重要的选择。原则：判断正在使用的措施和方法是否正确？发挥和创新的基础。

# 原则一：从和你有密切关联的外语内容学起

小孩子记不住上课老师说的内容？  
已学会/啰嗦 -> 无聊 -> 不 感兴趣 -> 无关内容 -> 保护自己的心情 -> 关闭耳朵，把注意力放到自己心中的话题，听不见外面的信息 -> 多年习惯变成条件反射 -> 只要遇到跟他无关的声音或信息，自觉走进听不见的状态。因为听不见，所以不明白，记不住。

大多数：  
信息跟自己无关，觉得信息没有重要意义。 因此不给予注意力。不给注意力的结果是记不住、学的慢甚至学不会。  
对自己有重要意义的信息，才能引起注意，给予足够的注意力。因此，学得快,甚至长期记忆。

研究结果：  
任何对个人有重要意义的事情或信息，自然会吸引一个高质量的注意力。因此能被快速记住。  
能意吸引注意力的事项，包括：爱情约会、生存相关、人生目标相关、生存意义相关。  
这些能让不自觉地投入足够的、高质量的注意力。当一门为 外语对个人的生存和未来有重要意义的时候，就有机会迅速学会。

-> 生死存亡、All in.  
答案 of 学不会的后果？ 将决定学外语的胜败。  
学外语跟个人生活或兴趣一点关系都没有？别学了。

第 1 ：为了把外语学得好、学得快，必须先找到这门外语对个人的重要价值和意义。  
第 2 ：为了准备每天学习的内容，也用“重要关联”的思路，作为选择一个学习素材的重要标准。  
每天选择自己有兴趣的学习内容。  
学习材料任意。

**原则 1 总结：  
为了把外语学好，要找这门外语跟自己的重要关联。在学习时，选择跟兴趣、动力等有重要关联的外语内容。**

# 原则二：把外语当成沟通工具

语言的本质是工具。使用语言的目的是沟通，得到沟通预期的结果。

1. 把外语当成工具，会你通过“用”来学外语，而不是先学后用。
   会一点，马上就去用。  
   每天经过真实的沟通和互动，增加一些新词汇、短句。在此基础上，继续扩大已沟通的范围和说话的委婉度。

2. 把外语当成工具，会让你功能出发，把形式放在后面 。  
   把注意力放在单词和词组的功能。先用功能明显的单词和词组来沟通，不让形式卡住学习的节点。  
   从功能出发，从沟通功能最强的含义单元开始学起的好处：  
   （1）几个关键单词和词组比带语法规矩的长句更容易被记住。  
   关键单词和词组，决定了一句话的核心含义。掌握了它们，能抓住整句话的大概意思。  
   在记住关键词和词组的前提下，有更多机会使用它们。 经过使用，成为长期记忆。同时在使用过程中，扩大沟通能力和沟通范围。  
   （2）功能明显的词和词组更容易更我们更有重要联系。  
   意识+潜意识 -> 更有重要联系的内容，给予足够的注意力 -> 记得更快。  
   （3）增强自信心 。  
   开口说话拿到预期的结果，哪怕说的根本不漂亮，但经过成功的沟通，能提高自信心。  
   -> 兴奋，不可思议。

3. 把外语当成工具，会让你更自然接受多种说法，因此不拘泥于一个标准答案。  
   **把注意力放在功能上，可以避免不必要的痛苦和挣扎，从而提高学习的效率。  
   不要特别在意别人的评价。**  
   “标准说法”会造成“哑巴英语”：让说话跟不上。  
   每一个意思，在任何语言通常有 3 个，甚至 8 个或更多不同的说法，每一个说话都能达到同一个沟通目的。因此，在这种情况下，寻找任何“唯一” 的准确说法是没有必要的。  
   错误的形式，也不影响沟通的效果，重要的词或词汇在就可以。  
   在成功沟通的基础上，把注意力放在形式的提高上。越能掌握外语的准确形式，越能被外国人看待为“自己人”。掌握外语沟通准确形式的前提，是基本上掌握了外语沟通的核心功能。

4. 把外语当成工具，可以获取“反馈”带来的巨大帮助，因此效果好。  
   **通过使用外语提高外语：反馈的力量会极大提高外语学习的速度。**  
   真实的沟通会口误、碰钉子，不要害怕丢脸，经过反馈，跟容易记牢。因为和“密切”关联原则有直接关系。因为关心沟通的结果，收到的反馈跟沟通结果有直接关系，因此会给予足够的注意力，提高记忆力。

**原则 2 的总结：  
语言本质上是一个沟通工具。是活的。当人们遇到一个新概念活新事物，人们自然会创造新的名称，来描述这些新概念和现象。**

# 原则三：理解了含义，自然能“获取”外语

## 1 可理解输入

可理解输入的定义：首先理解含义，然后下意识自动“获取”（无意中学会）外语的过程。

E.g., 不懂中文的老外，通过手势、表情、画小图片来沟通，从一开始什么都不懂，慢慢地知道对方在表达什么，没有刻意去学习和记忆单词，也至少掌握几个简单的词。  
可理解输入：外语学习时，当明白对方表达的含义时，无意中学会了对方正在使用的语言。

- 孩子是如何学会母语的？/ 小孩子为什么总是学习外语比成人快？  
  不是大脑与成人不同。最重要的差别是小孩接触新内容的过程，他们经常在一个“可理解输入”的环境，因此，潜意识可以不断吸收单词和说法。

  e.g., 妈妈说“吃饭”：同时动作模仿筷子把米饭塞到嘴里的动作，并且多次重复“吃饭”声音。这样整个环境，包括妈妈的动作都在沟通一个意思。因此，在已经明白意思的前提下，小孩在听到母语的声音，很快把“吃饭”这个声音符号和吃东西的概念结合在一起，最终学会了。

  e.g.,小孩子 3 岁左右，开始看动画片、漫画书等，他们用这种方式增强自己的语言基础。虽然不能全部听懂或看懂，但是基本明白意思。孩子不一定直接明白生词的确切含义，他们在懂整个上下文的情况下，大概感到一个陌生字或词是什么意思，接触多次后，明白了意思，自然就牢记了。

  e.g., 手势、表情、图片、看动画片、漫画书、声音、动作、实物

- **可理解输入 = 基本明白意思 = 明白上下文**

## 2 相信自己的潜意识学外语的能力

“可理解输入”的学习效果来自本能——潜意识的能力。

统计结果：在不同考试，“可理解输入”学习方法总是比传统学习方法效果好好。  
“可理解输入”学习方法 ： 潜意识地学习  
传统学习方法 ： 有意识地学习

| -              | 可理解输入 | 专注语法学习 |
| -------------- | ---------- | ------------ |
| 牛津语法测试   | 67.6       | 45.6         |
| 初级：阅读     | 22.5       | 14.5         |
| 初级：听力理解 | 24.9       | 17.45        |
| 初级：写作     | 19.4       | 7.5          |

潜意识地学习：效果快，不会忘记  
把注意力全部放在明白含义，放心地接受，在明白含义的基础上，自己的潜意识就会在背后努力支持你分析和记住。这样就能又快又踏实地学外语。

有意识地学习：效果慢，背单词需要画很长时间记忆，而且学会就忘。  
背单词、分析语法等

**原则 3 总结：  
先用一切办法理解含义，创造自然“获取”外语的条件。**

# 原则四：生理训练为主

大多人认为对外语学习的整体观点是累积“关于”外语的知识：背单词，学习语法。
累积这种知识对流利说英语帮助不大。

要学习外语，进行生理训练：大脑神经和外部肌肉。
与健美体操一样，练好外语的过程是一个肌肉训练的过程。

## 1 听觉过滤决定听力的效果。

听觉系统有专门过滤语言音节的部分：  
对熟悉的语言，大脑的过滤功能会把音节送到思维处理部。  
对不熟悉的语言音节，过滤把声音过滤掉，结果是听不熟悉的语言时，即会感到声音很模糊，甚至不知道自己在听什么。

e.g., 文莉从小在台湾学习英语，考试分数很高，但是刚去美国时听不懂、听不清英语，也不会说。  
哑巴英语——听不懂、听不清：在生理听力没问题，听觉系统过滤了不熟悉的语言音节。  
大脑对外语音节的过滤还没有健全地表现出来，因为英语音节的过滤不健全，导致她听不懂、听不清，所以没法说。

要把外语掌握好，一定要解决听力问题。

**如何解决听力问题？**  
是一个生理系统训练的过程  
累积单词、分析语法等，没有帮助。  
**用足够时间锻炼耳朵里的听觉神经。** 具体的锻炼方法在第三部分详细讲述。

## 2 你能感觉到自己的外语肌肉吗？

学好外语的重要前提是把发音练好。

如何解决发音问题？  
累积单词、分析语法等，没有帮助。  
和听力一样，是一个生理系统训练的过程。  
为了练好发音，必须练好面部肌肉。面部，包括舌头和嗓子，有 52 块不同的肌肉。

如何练习发音？  
发音练习和锻炼身体基本上是同一个规律。  
最好先用同一些方法松弛面部肌肉，然后模仿标准发音的模特，越练越接近标准即可。  
因为练习发音的时候是在练肌肉，刚开始身体有一些感觉和反应，比如酸疼，这是“练多了”的反应，是好事。

**和听力的训练一样，发音练习是肌肉训练的一个过程，因此要用体育训练的思路，指导自己找对路，这包括：经常练，在练习的过程中注意动作的准确度。**

## 3 创建外语条件反射

对方说“Good morning”，自然反应是“Good morning”。  
有很多日常的沟通是这一类的一问一答，也是有固定和半固定的形式，就像对联，即“日常对联”。  
别人的“日常对联”说到一半的时候，自己根本不用经过太多思考，就能马上采取合适的回应。这种对话属于大脑里面的条件反射，收到某一种刺激（信号）的时候，大脑系统会自动配备非常熟练的反应。

**要练好外语，除了听力和发音，也要故意把外语的条件反射练出来。**

**如何练习外语条件反射？  
多听多练“日常对联”。**
