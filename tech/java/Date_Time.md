# Date and Time

# 1 Java 8 API

- 日期和时间：LocalDate,LocalTime,LocalDateTime

- 时间差值：Period,ChronoUnit,Duration

  Period:  
  getYears（），getMonths（）和 getDays（）来计算

  ChronoUnit:  
   在单个时间单位内测量一段时间，例如天数或秒

  Duration:  
   基于时间的值（如秒，纳秒）测量时间量

- 时间戳:Instant

- 时间戳：System.currentTimeMillis()
