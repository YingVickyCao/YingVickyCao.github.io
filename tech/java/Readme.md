# Java

- Java 支持的四种类型：interface、class、数组（array）、基本类型（primitive）。  
  前三种是引用类型（reference type）。  
  类实例和数组是对象（oject）。

# 3 [类](./Class.md)

- 类的成员：field（域）、method（方法）、member class（成员类）、member interface（成员接口）
- 方法的签名（signature）：名称+所有参数类型组成，但不包括返回值。
- 继承（inheritance）：子类化（subclassing）。即子类继承父类。

# 4 接口
