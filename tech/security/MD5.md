# MD5

- MD5 = Message-Digest Algorithm 5（信息-摘要算法 5）
- MD5 值的唯一性  
  不管是一张图片，还是一个视频，任意一个数据，无论大小，都有一个唯一的 MD5 值，并且所有的 MD5 值长度一致，128 位，16 个字节
- 抗碰撞性也很强，伪造数据的难度很大
- MD5 算法是公开
- MD5 算法已被破解  
  2014 年中国山东大学的王小云教授公布破译了 MD5、HAVAL-128、 MD4 和 RIPEMD 算法的报告。通过加速的杂凑与冲撞方法破译了 MD5 算法。
- MD5 的前身有 MD2、MD3 和 MD4。
- MD5 是 其中一种摘要算法（哈希算法,Hash）, 不是加解密算法

## 优化

- 对 MD5 算法偏移  
  在原始数据上拼接一段数据在进行加密；或者根据原始数据分布范围进行转换，用得到的新值进行 MD5 计算，这样子被破解的风险就会大大降低。
