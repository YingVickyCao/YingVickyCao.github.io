- https vs http  
  | https | http |
  | ------ | -------------------------------------------------------------- |
  | 协议 | Hyper Text Transfer Protocol over SecureSocket Layer(SSL+HTTP) | Hyper Text Transfer Protocol |
  | 证书 | 申请证书 |
  | 安全性 | 加密 | 明文传输 |
  | 连接 | 有状态 | 无状态 |
  | 端口 | 443 | 80 |
